# UCLCHEM Notebooks
This directory contains the source code for the UCLCHEM notebooks as Python files
for improved code maintenance and flexibility. If you want to see the notebooks with
outputs present or as ready to go files:

## Full pre-run notebooks online
You can find pre-run versions with outputs of the notebooks on https://uclchem.github.io/tutorials/

## Locally on your system
You can use the command `(bash) notebooks.sh --generate` from the terminal to generate the files,
if you don't have everything installed, it will walk you through it.

## Downloading from the release artifacts
UCLCHEM pre-runs the notebooks on Github, you can easily download the zip from the latest release:
https://github.com/uclchem/UCLCHEM/releases/latest
